Asta Smart Parking

Built With

Laravel Bootstrap MySQL

Asta Smart Parking is a smart parking system that combines number plate recognition (LPR) technology and unmanned automatic gates to create an efficient and contactless parking experience. The system is equipped with features such as membership, a mobile app for officers and members, and a web admin for recording transactions and real-time reports. Currently, Asta Smart Parking is used across all Hermina Hospital locations as a reliable and integrated modern parking solution.
