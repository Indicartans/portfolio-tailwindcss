Sipenting — Stunting Detection

Built With

Laravel Bootstrap MySQL

Sipenting (Stunting Detection Expert System) is a web-based application designed to help detect the possibility of stunting in children. This application uses the Certainty Factor (CF) method to analyse the symptoms entered by users, then calculates the level of certainty regarding the possibility of stunting.

With a user-friendly interface and quick process, Sipenting provides diagnostic results along with suggestions or recommendations for actions that can be taken. The application aims to raise awareness and simplify the process for the public and healthcare workers to conduct early detection of stunting independently and efficiently.
